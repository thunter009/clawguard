# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 0.1.x   | Yes                |

## Reporting a Vulnerability

ClawGuard is a security tool. If you discover a vulnerability in ClawGuard
itself, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, please email: **flydevox@gmail.com** 

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge receipt within 48 hours and aim to release a fix within
7 days for critical issues.

## Scope

This policy covers vulnerabilities in ClawGuard's own code, including:
- Bypasses of origin validation
- Scanner evasion techniques
- Cost limiter circumvention
- Configuration parsing flaws
- Log injection

This policy does NOT cover:
- Vulnerabilities in OpenClaw itself (report to OpenClaw's security team)
- Vulnerabilities in upstream dependencies (report to the respective project)
